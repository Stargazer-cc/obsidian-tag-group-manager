# 悬浮面板颜色支持

## 功能说明

现在悬浮面板（HierarchyBoard）中的标签也能应用颜色设置了！

## 实现内容

### 1. 彩虹预设颜色
当标签设置了预设颜色（红、蓝、绿、橙、紫、青、粉）时：
- 应用 `tag-group-manager-rainbow-tag` 类
- 使用 `data-color` 属性存储颜色值
- 显示柔和的渐变背景

### 2. 自定义颜色
当标签设置了自定义颜色（十六进制）时：
- 应用 `tgm-custom-color-tag` 类
- 计算 RGB 值并生成渐变背景
- 背景：`rgba(r, g, b, 0.08)` 到 `rgba(r, g, b, 0.06)`
- 文字：`rgba(r, g, b, 0.85)`
- 边框：`rgba(r, g, b, 0.15)`

### 3. 默认样式
当启用了颜色设置但标签没有设置颜色时：
- 应用 `tgm-default-rainbow-tag` 类
- 使用中性的灰蓝色（148, 163, 184）
- 保持统一的渐变风格

### 4. 未启用颜色设置
当未启用颜色设置时：
- 使用默认的 pill 样式
- 背景：`var(--background-secondary-alt)`
- 无特殊颜色效果

## 代码实现

### 颜色检测和应用
```typescript
if (this.plugin.settings.enableCustomColors) {
    const customColor = this.plugin.settings.tagColors[node.fullName] || null;
    if (customColor) {
        const isRainbowColor = customColor.startsWith('var(--color-');
        if (isRainbowColor) {
            // 应用彩虹预设颜色
            pillEl.addClass('tag-group-manager-rainbow-tag');
            pillEl.setAttribute('data-color', customColor);
        } else {
            // 应用自定义颜色
            pillEl.addClass('tgm-custom-color-tag');
            // 计算 RGB 并设置渐变...
        }
    } else {
        // 应用默认样式
        pillEl.addClass('tgm-default-rainbow-tag');
    }
}
```

### RGB 转换
```typescript
const hexToRgb = (hex: string) => {
    const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
    return result ? {
        r: parseInt(result[1], 16),
        g: parseInt(result[2], 16),
        b: parseInt(result[3], 16)
    } : null;
};
```

### 渐变背景
```typescript
const bgColor = `rgba(${rgb.r}, ${rgb.g}, ${rgb.b}, 0.08)`;
const textColor = `rgba(${rgb.r}, ${rgb.g}, ${rgb.b}, 0.85)`;
const borderColor = `rgba(${rgb.r}, ${rgb.g}, ${rgb.b}, 0.15)`;

pillEl.style.setProperty('background', 
    `linear-gradient(145deg, ${bgColor}, rgba(${rgb.r}, ${rgb.g}, ${rgb.b}, 0.06))`, 
    'important');
```

## 注意事项

### 1. 性能
- 颜色计算在渲染时进行
- 使用内联样式（`style.setProperty`）
- 对性能影响很小

### 2. 兼容性
- 与现有的 CSS 类兼容
- 不影响未设置颜色的标签
- 可以与其他样式共存

### 3. 主题适配
- 使用 RGBA 颜色，自动适配深色/浅色主题
- 透明度经过优化，确保可读性
- 边框和文字颜色自动计算
