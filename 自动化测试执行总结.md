## 使用的Obsidian CLI命令

### 1. 创建测试文件
```bash
obsidian create name="测试标签库" content="..." silent overwrite
```

### 2. 查看标签统计
```bash
obsidian tags sort=name counts
```

### 3. 访问插件并执行操作
```bash
obsidian eval code="const plugin = app.plugins.plugins['tag-group-manager']; ..."
```

### 4. 创建标签组
```bash
obsidian eval code="plugin.settings.tagGroups.push({...}); plugin.saveSettings();"
```

### 5. 配置规则
```bash
obsidian eval code="plugin.settings.autoAddRules = '...'; plugin.saveSettings();"
```

### 6. 执行自动添加
```bash
obsidian eval code="plugin.autoScanTagsToGroups().then(count => {...});"
```

### 7. 验证结果
```bash
obsidian eval code="JSON.stringify(plugin.settings.tagGroups.filter(...));"
```

### 8. 清理测试数据
```bash
obsidian eval code="plugin.settings.tagGroups = plugin.settings.tagGroups.filter(...);"
```
