# 简洁边框设计

## 设计原则

只通过宽度和颜色来区分外边框和树形连接线，不使用透明度。

## 视觉元素

### 1. 外边框（看板容器）
- **宽度**：2px
- **颜色**：
  - 深色主题：`rgba(255, 255, 255, 0.15)`
  - 浅色主题：`rgba(0, 0, 0, 0.12)`
- **特点**：较粗，较明显

### 2. 树形连接线（垂直线 + 水平线）
- **宽度**：1px
- **颜色**：`var(--background-modifier-border)`
- **悬停**：`var(--text-muted)`
- **特点**：较细，使用主题边框色

### 3. 圆点标记
- **大小**：4px（静态），6px（悬停）
- **颜色**：`var(--text-muted)`（静态），`var(--interactive-accent)`（悬停）
- **特点**：悬停时变大变亮并发光

## 视觉对比

| 元素 | 宽度 | 颜色 | 视觉权重 |
|------|------|------|----------|
| 外边框 | 2px | 较明显 | 重 |
| 树形线 | 1px | 主题边框色 | 轻 |

## 视觉效果

## CSS 实现

```css
/* 外边框 - 2px */
.theme-dark .tgm-hierarchy-board {
  border: 2px solid rgba(255, 255, 255, 0.15);
}

/* 树形连接线 - 1px */
.tgm-hb-children {
  border-left: 1px solid var(--background-modifier-border);
}

.tgm-hb-content-wrapper::before {
  height: 1px;
  background: var(--background-modifier-border);
}

/* 悬停时变深 */
.tgm-hb-item:hover > .tgm-hb-children {
  border-left-color: var(--text-muted);
}

.tgm-hb-content-wrapper:hover::before {
  background: var(--text-muted);
}
```

## 设计优势

✅ **简洁清晰** - 只用宽度和颜色区分，无复杂的透明度
✅ **主次分明** - 外边框粗（2px），树形线细（1px）
✅ **视觉统一** - 树形线使用主题边框色，自动适配
✅ **交互友好** - 悬停时颜色变深，提供视觉反馈
