# 测试批量颜色功能

## 功能说明

新增了整组标签批量添加颜色的功能，位于标签组设置区域。

## 使用方法

1. 打开插件设置页面
2. 确保"启用自定义标签颜色"开关已打开
3. 在任意标签组的头部，你会看到一个新的 🎨 按钮
4. 点击该按钮，会打开颜色选择器
5. 选择一个颜色后，该组的所有标签都会应用这个颜色

## 使用 Obsidian CLI 测试

### 1. 重载插件
```bash
obsidian plugin:reload id=tag-group-manager
```

### 2. 检查错误
```bash
obsidian dev:errors
```

### 3. 截图验证
```bash
obsidian dev:screenshot path=group-color-test.png
```

### 4. 打开设置页面（通过 eval）
```bash
obsidian eval code="app.setting.open(); app.setting.openTabById('tag-group-manager')"
```

## 功能特点

- 🎨 一键为整组标签设置相同颜色
- ✅ 支持预设颜色和自定义颜色
- 🔄 实时刷新显示
- 💾 自动保存设置
- 🌍 支持中英文界面
