# 极简设计测试清单

## 快速验证

### 1. 重载插件
```bash
obsidian plugin:reload id=tag-group-manager
```

### 2. 检查错误
```bash
obsidian dev:errors
```

### 3. 视觉检查

**应该看到：**
- ✅ 只有灰色边框（1px 实线）
- ✅ 简洁的圆点标记（4px）
- ✅ 清晰的缩进层级

**不应该看到：**
- ❌ 黄色渐变线
- ❌ 多种颜色的边框
- ❌ 复杂的连接线

### 4. 悬停测试

**悬停在节点上应该看到：**
- ✅ 边框颜色变深
- ✅ 圆点变大发光（6px，主题色）
- ✅ Pill 向右移动并放大

### 5. 截图对比
```bash
obsidian dev:screenshot path=final-minimal-design.png
```

## 验收标准

- ✅ 视觉极简（只有灰线和圆点）
- ✅ 层级清晰（缩进明显）
- ✅ 交互流畅（悬停反馈）
- ✅ 无功能退化

## 完成！🎉

如果以上都通过，说明极简设计成功实现！
