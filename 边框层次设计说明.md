# 边框层次设计说明

## 设计理念

通过不同的边框样式，清晰区分看板的外边框和内部的树形连接线。

## 视觉层次

### 1. 外边框（看板容器）
**作用**：定义看板的边界，突出整个面板

**样式：**
- 宽度：2px（较粗）
- 颜色：
  - 深色主题：`rgba(255, 255, 255, 0.15)` - 较亮的白色
  - 浅色主题：`rgba(0, 0, 0, 0.12)` - 较深的黑色
- 圆角：12px
- 阴影：`0 8px 32px 0 rgba(0, 0, 0, 0.2)`

**特点：**
- ✅ 较粗（2px）
- ✅ 较明显（透明度更高）
- ✅ 有圆角和阴影
- ✅ 突出整个面板

### 2. 树形连接线（内部结构）
**作用**：表示层级关系，引导视线

**样式：**
- 宽度：1px（较细）
- 颜色：`var(--background-modifier-border)` - 主题边框色
- 透明度：60%（静态），80%（悬停）
- 无圆角，无阴影

**特点：**
- ✅ 较细（1px）
- ✅ 较淡（60% 透明度）
- ✅ 简洁细腻
- ✅ 不喧宾夺主

## 视觉对比

### 外边框 vs 树形连接线

| 属性 | 外边框 | 树形连接线 |
|------|--------|------------|
| 宽度 | 2px | 1px |
| 透明度 | 较高（15%/12%） | 较低（60%） |
| 视觉权重 | 重 | 轻 |
| 作用 | 定义边界 | 表示层级 |

## CSS 实现

### 外边框
```css
/* 深色主题 */
.theme-dark .tgm-hierarchy-board {
  border: 2px solid rgba(255, 255, 255, 0.15);
}

/* 浅色主题 */
.theme-light .tgm-hierarchy-board {
  border: 2px solid rgba(0, 0, 0, 0.12);
}
```

### 树形连接线
```css
/* 垂直线 */
.tgm-hb-children {
  border-left: 1px solid var(--background-modifier-border);
  opacity: 0.6;
}

/* 水平线 */
.tgm-hb-content-wrapper::before {
  height: 1px;
  background: var(--background-modifier-border);
  opacity: 0.6;
}
```

## 设计优势

### 1. 清晰的视觉层次
- ✅ 外边框粗而明显，定义面板边界
- ✅ 树形线细而淡雅，表示内部结构
- ✅ 两者形成鲜明对比

### 2. 符合视觉习惯
- ✅ 外边框像"画框"，突出整体
- ✅ 树形线像"引导线"，辅助理解
- ✅ 主次分明，易于理解

### 3. 美观协调
- ✅ 外边框有圆角和阴影，精致感
- ✅ 树形线简洁细腻，不抢眼
- ✅ 整体和谐统一

### 4. 交互友好
- ✅ 悬停时树形线变深（60% → 80%）
- ✅ 外边框保持稳定
- ✅ 清晰的视觉反馈

## 固定状态特殊样式

当看板被固定时，外边框会变为主题色：

```css
.tgm-hierarchy-board.pinned {
  border-color: var(--interactive-accent);
  box-shadow: 0 8px 32px 0 rgba(var(--interactive-accent-rgb), 0.2);
}
```

**效果：**
- 边框变为主题色（如蓝色、紫色等）
- 阴影也带有主题色
- 明确表示"已固定"状态
