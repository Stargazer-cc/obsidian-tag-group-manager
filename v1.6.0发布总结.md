### 核心算法

#### 1. 标签显示过滤算法
```typescript
function shouldHideTag(
  tag: string,
  allTags: string[],
  autoExpandedTags?: string[],
  minDepth: number = 3
): boolean {
  // 1. 检查是否自动添加
  if (!autoExpandedTags?.includes(tag)) return false;
  
  // 2. 检查是否多级标签
  if (!tag.includes('/')) return false;
  
  // 3. 检查深度
  const depth = tag.split('/').length;
  if (depth < minDepth) return false;
  
  // 4. 检查是否有子节点
  const hasChildren = allTags.some(t => t.startsWith(tag + '/'));
  
  // 5. 叶子节点需要隐藏
  return !hasChildren;
}
```

#### 2. 展开深度判断算法
```typescript
function getGroupExpandDepth(
  group: TagGroup,
  tagGroupSets: TagGroupSet[]
): number {
  if (!group.id) return 2;
  
  // 在组集中 = 3级展开
  // 不在组集中 = 2级展开
  const inSet = tagGroupSets.some(
    set => set.groupIds.includes(group.id!)
  );
  return inSet ? 3 : 2;
}
```

#### 3. Live Preview样式应用算法
```typescript
private processTagElement(element: HTMLElement) {
  // 1. 只处理标签文本部分
  if (!element.classList.contains('cm-hashtag-end')) return;
  
  // 2. 避免重复处理
  if (element.hasAttribute('data-tag-name')) return;
  
  // 3. 获取标签文本
  const tagText = element.textContent?.trim();
  if (!tagText) return;
  
  // 4. 检查是否有配置颜色
  if (this.settings.tagColors[tagText]) {
    // 5. 添加data属性
    const safeTagAttr = tagText.replace(/[\/\s]/g, '-');
    element.setAttribute('data-tag-name', safeTagAttr);
    
    // 6. 处理#号元素
    const prevSibling = element.previousElementSibling;
    if (prevSibling?.classList.contains('cm-hashtag-begin')) {
      prevSibling.setAttribute('data-tag-name', safeTagAttr);
    }
  }
}
```

### 优化措施
1. **MutationObserver优化**
   - 只监听childList变化
   - 避免重复处理

## 🎓 用户指南

### 快速开始

#### 1. 使用多级标签展开
```
步骤1：打开设置 → 多级标签适配
步骤2：选择展开深度（2级或3级）
步骤3：（可选）输入要展开的第一级标签
步骤4：点击"⚡ 展开多级标签"
步骤5：查看生成的标签组和组集
```

#### 2. 启用Live Preview标签颜色
```
步骤1：打开设置 → 颜色设置
步骤2：启用"启用自定义标签颜色"
步骤3：启用"将标签颜色应用到正文中"
步骤4：为标签设置颜色
步骤5：在Live Preview模式查看效果
```

#### 3. 使用自动添加标签
```
步骤1：打开设置 → 自动添加标签至标签组
步骤2：启用"自动添加标签"
步骤3：设置规则（标签名:标签组名）
步骤4：点击"扫描并添加"
步骤5：查看添加结果
```

### 常见问题

**Q: 展开后的标签组可以修改吗？**
A: 可以。展开后的标签组和普通标签组一样，可以自由修改、删除、添加标签。

**Q: 重复展开会覆盖已有配置吗？**
A: 不会。重复展开会自动添加序号，不会覆盖已有的标签组和组集。

**Q: Live Preview颜色不显示怎么办？**
A: 确保启用了"启用自定义标签颜色"和"将标签颜色应用到正文中"两个开关。

**Q: 自动添加的标签可以删除吗？**
A: 可以。删除后不会被重新添加，插件会记住你的删除操作。
