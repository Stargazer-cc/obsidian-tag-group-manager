# 经典树形结构设计

## 设计说明

现在使用经典的树形结构，清晰区分两种连接线：

### 1. 垂直连接线（展开线）
- **作用**：连接父节点和所有子节点
- **样式**：1px 灰色实线
- **位置**：子节点容器的左侧
- **长度**：覆盖所有子节点的高度

### 2. 水平连接线（分支线）
- **作用**：从垂直线连接到每个子节点
- **样式**：1px 灰色实线
- **位置**：从垂直线到圆点（8px 长）
- **特点**：每个节点都有独立的水平线

### 3. 圆点标记
- **作用**：标记节点位置
- **样式**：4px 圆点
- **位置**：水平线的末端
- **效果**：悬停时变大发光

**说明：**

- `├─` 或 `└─` = 水平连接线（分支线）
- `●` = 圆点标记

## CSS 实现

### 垂直连接线
```css
.tgm-hb-children {
  border-left: 1px solid var(--background-modifier-border);
}
```

### 水平连接线
```css
.tgm-hb-content-wrapper::before {
  content: '';
  width: 8px;
  height: 1px;
  background: var(--background-modifier-border);
}
```

### 圆点标记
```css
.tgm-hb-content-wrapper::after {
  content: '';
  width: 4px;
  height: 4px;
  background: var(--text-muted);
  border-radius: 50%;
}
```

## 悬停效果

### 静态状态
- 垂直线：浅灰色（`--background-modifier-border`）
- 水平线：浅灰色（`--background-modifier-border`）
- 圆点：灰色，50% 透明度

### 悬停状态
- 垂直线：深灰色（`--text-muted`）
- 水平线：深灰色（`--text-muted`）
- 圆点：主题色，100% 透明度 + 发光

## 设计优势

### 1. 清晰的视觉层次
- ✅ 垂直线表示"这是一组子节点"
- ✅ 水平线表示"这是其中一个子节点"
- ✅ 圆点标记"这是节点本身"

### 2. 经典易懂
- ✅ 符合传统树形结构的视觉习惯
- ✅ 用户一眼就能理解层级关系
- ✅ 类似文件管理器的树形视图

### 3. 简洁美观
- ✅ 只使用灰色系，视觉统一
- ✅ 线条细腻（1px），不会显得笨重
- ✅ 圆点小巧（4px），不会喧宾夺主

### 4. 交互友好
- ✅ 悬停时所有连接线变深
- ✅ 圆点发光提供视觉反馈
- ✅ 清晰表明当前悬停的节点

## 技术细节

### 使用伪元素
- `::before` - 水平连接线
- `::after` - 圆点标记
- 不增加 DOM 节点，性能更好

### 位置计算
```css
/* 水平线：从垂直线（left: -12px）到圆点（8px 长） */
.tgm-hb-content-wrapper::before {
  left: -12px;
  width: 8px;
}

/* 圆点：在水平线末端（left: -4px） */
.tgm-hb-content-wrapper::after {
  left: -4px;
}
```

### 层叠顺序
```css
/* 圆点在水平线上方 */
.tgm-hb-content-wrapper::after {
  z-index: 1;
}
```
