# 层级看板最终简化设计

## 设计理念：极简主义

> "简洁是终极的复杂" - 达芬奇

只使用两个核心元素：
1. **灰色边框** - 表示层级关系
2. **圆点标记** - 标记节点位置

## 视觉效果

### 静态状态
```
插件
│
├─ ● TagGroupManager
│  │
│  ├─ ● 实现
│  │  ├─ ● 标签解析
│  │  ├─ ● 多级展开
│  │  └─ ● UI
│  │     └─ ● 折叠逻辑
│  │
│  └─ ● 需求
│     ├─ ● 历史遗留
│     └─ ● 用户反馈
```

**特点：**
- ✅ 只有灰色边框（`border-left: 1px solid`）
- ✅ 简洁的圆点标记（4px）
- ✅ 清晰的缩进层级
- ✅ 极简设计，无视觉噪音

### 悬停状态
- 边框颜色变深（`--background-modifier-border` → `--text-muted`）
- 圆点变大发光（4px → 6px，主题色）
- Pill 向右移动并放大

## CSS 实现

### 简洁的边框
```css
.tgm-hb-children {
  margin-left: 16px;
  margin-top: 4px;
  padding-left: 12px;
  border-left: 1px solid var(--background-modifier-border);
}

/* 悬停时边框变深 */
.tgm-hb-item:hover > .tgm-hb-children {
  border-left-color: var(--text-muted);
}
```

### 圆点标记
```css
.tgm-hb-content-wrapper::before {
  content: '';
  width: 4px;
  height: 4px;
  background: var(--text-muted);
  border-radius: 50%;
  opacity: 0.4;
}

/* 悬停时圆点发光 */
.tgm-hb-content-wrapper:hover::before {
  width: 6px;
  height: 6px;
  background: var(--interactive-accent);
  opacity: 1;
  box-shadow: 0 0 4px var(--interactive-accent);
}
```

## 设计优势

### 1. 极简清晰
- ❌ 去掉了渐变边框（黄色线）
- ✅ 只保留必要的灰色边框
- ✅ 视觉更加统一和简洁

### 2. 层级清晰
- 灰色边框清晰表示父子关系
- 缩进表示层级深度
- 圆点标记节点位置

### 3. 交互友好
- 悬停时边框变深，增强连接感
- 圆点发光，提供视觉反馈
- Pill 移动，暗示可点击

### 4. 性能优良
- 使用简单的 border 属性
- 无复杂的渐变计算
- 渲染性能更好

## 对比分析

### 优化前（传统树形线）
```
├── 节点1
│   ├── 子节点1
│   │   ├── 孙节点1
│   │   └── 孙节点2
│   └── 子节点2
└── 节点2
```
- ❌ L 形连接线复杂
- ❌ 多层级时杂乱

### 中间方案（渐变线 + 灰线）
```
│ (渐变黄色)
├─ (灰色)
```
- ⚠️ 两种线显得冗余
- ⚠️ 不够简约

### 最终方案（只有灰线 + 圆点）
```
│ (灰色)
├─ ● (圆点)
```
- ✅ 极简清晰
- ✅ 视觉统一
- ✅ 易于理解

## 视觉层次

### 通过以下元素表现层级：

1. **缩进**（主要）
   - 每层增加 28px
   - 最直观的层级表现

2. **边框**（辅助）
   - 连接父子节点
   - 引导视线向下

3. **圆点**（标记）
   - 标记节点位置
   - 提供视觉锚点

4. **透明度**（深度）
   - 深层节点略微透明
   - 表现视觉深度

5. **字重**（重要性）
   - 根节点使用粗体
   - 突出顶层结构

## 颜色系统

### 静态状态
- 边框：`var(--background-modifier-border)` - 主题边框色
- 圆点：`var(--text-muted)` - 主题静音色，40% 透明度
- Pill：`var(--background-secondary-alt)` - 主题次要背景色

### 悬停状态
- 边框：`var(--text-muted)` - 变深
- 圆点：`var(--interactive-accent)` - 主题强调色，100% 透明度
- Pill：`var(--interactive-accent)` - 主题强调色背景

### 主题适配
- 浅色主题：灰色边框和圆点
- 深色主题：浅灰色边框和圆点
- 自动适配，无需额外配置

- 渲染快速
